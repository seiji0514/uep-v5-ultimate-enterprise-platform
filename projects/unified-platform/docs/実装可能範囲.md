## 実装優先度（小工数で効果大）

1. **API 認証必須化** … 機密向上、影響範囲は限定的
2. **構造化ログ** … 精密・運用性向上
3. **PC 表示最適化** … レイアウト改善、即効性あり
4. **デザインシステム（余白・フォント）** … 全体の一貫性

## 実装困難（大工数 or 外部依存）

- MFA、アカウントロック … 認証基盤の大幅変更
- 本格的な RBAC … ロール・権限テーブル設計が必要
