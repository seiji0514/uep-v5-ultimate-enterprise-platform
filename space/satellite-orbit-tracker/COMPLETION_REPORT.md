## 🎯 技術的特徴

### 1. 軌道力学計算

**Kepler軌道要素**:
```python
{
    'semi_major_axis': 6791.0,  # 軌道長半径 (km)
    'eccentricity': 0.0003,     # 離心率
    'inclination': 51.6,        # 軌道傾斜角 (degrees)
    'raan': 0.0,                # 昇交点赤経 (degrees)
    'arg_perigee': 0.0,         # 近地点引数 (degrees)
    'mean_anomaly': 0.0         # 平均近点角 (degrees)
}
```

**計算フロー**:
1. 平均近点角の時間発展
2. Kepler方程式を解く（ニュートン法）
3. 離心近点角 → 真近点角の変換
4. ECI座標系での位置・速度計算
5. GMST計算 → ECEF変換
6. 地理座標（緯度・経度・高度）変換

### 2. 座標変換

**ECI (Earth-Centered Inertial)**:
- 慣性座標系（地球の自転を無視）
- 軌道計算の標準座標系

**ECEF (Earth-Centered Earth-Fixed)**:
- 地球固定座標系
- 地球と一緒に回転

**地理座標**:
- 緯度（latitude）
- 経度（longitude）
- 高度（altitude）

### 3. 3D可視化

**Three.js実装**:
- 地球: `SphereGeometry(6371, 64, 64)` (半径6371km)
- 衛星: `SphereGeometry(100, 16, 16)` (可視化のため拡大)
- 軌道線: `LineGeometry` (24時間分の軌跡)
- ライト: Ambient + Directional
- カメラ: Perspective (45° FOV)

## 📝 技術的な補足

### Kepler方程式とは？

衛星の位置を求めるための基本方程式:

```
M = E - e*sin(E)
```

- M: 平均近点角（時間に比例）
- E: 離心近点角（ニュートン法で解く）
- e: 離心率

この方程式を解くことで、衛星の正確な位置を計算できる。

### ニュートン法による解法

```python
def _solve_kepler(self, M: float, e: float) -> float:
    E = M  # 初期値
    for _ in range(100):
        f = E - e * np.sin(E) - M
        f_prime = 1 - e * np.cos(E)
        E_new = E - f / f_prime
        if abs(E_new - E) < 1e-8:
            return E_new
        E = E_new
    return E
```

収束判定: `|E_new - E| < 1e-8`（精度8桁）
