```javascript
// 複数衛星間の衝突予測
calculateClosestApproach(orbit1, orbit2) {
    let minDistance = Infinity;
    for (let i = 0; i < orbit1.length; i++) {
        const distance = Math.sqrt(
            (p1.x - p2.x)**2 + 
            (p1.y - p2.y)**2 + 
            (p1.z - p2.z)**2
        );
        if (distance < minDistance) {
            minDistance = distance;
            minTime = orbit1[i].timestamp;
        }
    }
    return { minDistance, minTime };
}
```

## 📊 技術的詳細

### SGP4モデル（Simplified General Perturbations 4）

NASAが開発した標準軌道伝播モデル:

**考慮する摂動**:
1. **J2摂動**（地球の扁平率）
2. **大気抵抗**（LEO衛星）
3. **太陽輻射圧**
4. **月・太陽の引力**

**精度**:
- 短期予測（1-3日）: 誤差 < 500m
- 中期予測（1週間）: 誤差 < 5km

### Isolation Forest（異常検知AI）

**アルゴリズム**:
1. ランダムに特徴量を選択
2. ランダムに分割点を選択
3. 決定木を構築
4. 分離に必要な深さで異常度を判定

**特徴量**:
```javascript
features = [
    mean(altitudes),      // 高度平均
    std(altitudes),       // 高度標準偏差
    mean(speeds),         // 速度平均
    std(speeds),          // 速度標準偏差
    max(alt) - min(alt)   // 高度変動範囲
]
```

**異常判定**:
- 異常スコア < -0.5 → 異常
- 信頼度 = (1 + anomalyScore) × 100%
