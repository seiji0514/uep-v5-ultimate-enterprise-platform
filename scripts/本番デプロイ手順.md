## 本番デプロイ時の手順

### 1. ドメインを設定

```bash
python scripts/set-production-domain.py your-domain.com
```

例: `uep.mycompany.co.jp` の場合
```bash
python scripts/set-production-domain.py uep.mycompany.co.jp
```

→ API: https://api.uep.mycompany.co.jp  
→ フロント: https://uep.mycompany.co.jp, https://app.uep.mycompany.co.jp

### 2. Sentry を設定（推奨）

1. [sentry.io](https://sentry.io) でプロジェクト作成
2. DSN を取得
3. `frontend/.env.production` の `REACT_APP_SENTRY_DSN=` に設定

### 3. フロントエンドビルド

```bash
cd frontend
set NODE_OPTIONS=--max-old-space-size=8192
npm run build
```

### 4. バックエンド起動

```bash
cd backend
python main.py
```

### 5. デプロイ

- フロント: `frontend/build` を静的ホスト（Nginx, S3, Vercel 等）に配置
- バックエンド: Gunicorn/Uvicorn で起動、リバースプロキシ経由で HTTPS 化

## ローカルでの本番ビルド確認

### 方法1: 一括起動

```
start-production-all.bat
```

→ バックエンド（別ウィンドウ）＋フロントエンド（ビルド＋起動）

### 方法2: 個別起動

```
REM ターミナル1: バックエンド
start-backend-production.bat

REM ターミナル2: フロントエンド（ビルド＋起動）
start-frontend-production.bat
```

→ http://localhost:3000 で本番ビルドを確認
