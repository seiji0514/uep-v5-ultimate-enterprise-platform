# HTML版 - 完全クライアントサイド

## 【概要】

**HTMLファイル1つだけで動作する、最もシンプルなバージョンです。**

- サーバー不要
- データベース不要
- Node.js不要
- Python不要
- ブラウザだけで動作

## 【使い方】

1. `index.html` をブラウザで開くだけ
2. データはブラウザのローカルストレージに保存されます

## 【機能】

✅ IoTセンサー管理  
✅ 環境データ登録・表示  
✅ 交通データ登録・表示  
✅ エネルギーデータ登録・表示  
✅ ESGレポート生成  
✅ ダッシュボード表示  

## 【データ保存】

- データはブラウザのローカルストレージに保存されます
- ブラウザの設定をクリアするとデータも消えます
- データのエクスポート/インポート機能は今後追加予定

## 【対応ブラウザ】

- Chrome（推奨）
- Firefox
- Safari
- Edge

## 【制限事項】

- データはローカルストレージに保存されるため、ブラウザを変えるとデータは共有されません
- 大規模なデータには不向き（ローカルストレージの容量制限あり）
- 複数ユーザーでの同時利用は不可
