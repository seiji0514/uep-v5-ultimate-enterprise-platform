# 分野別使用例

## 🎯 概要

AI自動音声応答システムを様々な分野で使用する具体的な例

## 📋 分野別の使用例

### 1. 医療・ヘルスケア分野

#### 使用例1: 症状相談

```
ユーザー: 「頭痛が続いています」
↓
システム: 医療知識ベースから検索
↓
応答: 「頭痛が続く場合は、以下の可能性があります。
       1. 緊張型頭痛
       2. 片頭痛
       3. その他の原因
       
       症状が続く場合は、医療機関を受診することをお勧めします。」
（音声で応答）
```

#### 使用例2: 健康管理

```
ユーザー: 「今日の歩数は？」
↓
システム: 健康管理アプリからデータを取得
↓
応答: 「今日の歩数は8,500歩です。目標の10,000歩まであと1,500歩です。」
（音声で応答）
```

## 🔧 分野モードの設定方法

### WebSocket経由で設定

```javascript
// 医療分野モードを有効化
ws.send(JSON.stringify({
    type: "set_domain_mode",
    domain: "healthcare"
}));

// 法務分野モードを有効化
ws.send(JSON.stringify({
    type: "set_domain_mode",
    domain: "legal"
}));

// 金融分野モードを有効化
ws.send(JSON.stringify({
    type: "set_domain_mode",
    domain: "finance"
}));
```

### API経由で設定

```bash
POST http://localhost:8001/api/domain/healthcare/set
{
  "connection_id": 12345
}
```

### 利用可能な分野を確認

```bash
GET http://localhost:8001/api/domain/list
```

## 🎯 実用的な使用フロー

### フロー1: 医療分野での使用

```
1. 医療分野モードを有効化
   → set_domain_mode("healthcare")

2. 症状を音声で説明
   → 「頭痛が続いています」

3. AIが医療知識ベースから検索
   → 関連情報を取得

4. 応答を音声で返す
   → 「頭痛が続く場合は...」
```

### フロー2: 法務分野での使用

```
1. 法務分野モードを有効化
   → set_domain_mode("legal")

2. 法務に関する質問を音声で
   → 「この契約条項のリスクは？」

3. AIが契約書を分析
   → リスクを検出

4. 応答を音声で返す
   → 「この契約書には以下のリスクが...」
```

## 📝 まとめ

AI自動音声応答システムは、以下の様々な分野で活用できます：

1. **医療・ヘルスケア** - 症状相談、健康管理
2. **法務・法律** - 契約書分析、法務Q&A
3. **金融・FinTech** - 不正検知、リスク評価
4. **教育・学習** - 学習支援、クイズ生成
5. **製造業** - 品質管理、異常検知
6. **小売・EC** - 商品検索、レコメンデーション
7. **不動産** - 物件検索、価格情報
8. **農業** - 病害虫診断、収穫予測
9. **交通・物流** - 経路検索、物流管理
10. **エンターテイメント** - コンテンツ検索、レコメンデーション

各分野の既存システムと統合することで、**分野特化の音声アシスタント**として活用できます。
