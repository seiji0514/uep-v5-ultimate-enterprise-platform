## 📋 実装する機能

### Phase 1: コアシステムの統合（1週間）

#### 1.1 統合APIゲートウェイ

```python
# backend/api/gateway.py
class UnifiedAPIGateway:
    """統合APIゲートウェイ"""
    
    async def process_voice_request(
        self,
        audio_chunk: bytes,
        domain: str,
        connection_id: int
    ):
        """
        音声リクエストを処理
        
        Args:
            audio_chunk: 音声データ
            domain: 分野（healthcare, legal, finance, etc.）
            connection_id: 接続ID
        """
        # 1. 音声認識
        text = await self.voice_service.transcribe(audio_chunk)
        
        # 2. 分野別モジュールで処理
        response = await self.domain_modules[domain].process(text)
        
        # 3. 音声合成
        audio = await self.voice_service.text_to_speech(response)
        
        return audio, response
```

#### 1.2 分野別モジュール統合

```python
# backend/modules/domain_modules.py
class DomainModuleManager:
    """分野別モジュール管理"""
    
    def __init__(self):
        self.modules = {
            "healthcare": HealthcareModule(),
            "legal": LegalModule(),
            "finance": FinanceModule(),
            "education": EducationModule(),
            "manufacturing": ManufacturingModule(),
            "retail": RetailModule(),
            # ... その他の分野
        }
    
    async def process(self, domain: str, text: str):
        """分野別処理"""
        if domain in self.modules:
            return await self.modules[domain].handle_query(text)
        else:
            return "未対応の分野です。"
```
