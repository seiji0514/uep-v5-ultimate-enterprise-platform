# 既存システム以外の活用方法

## 🎯 概要

AI自動音声応答システムを、既存の面談準備資料以外で活用する方法

## 🚀 主な活用方法

### 1. RAG（検索拡張生成）システムとの統合

#### 概要
リアルタイムで知識ベースから情報を検索し、正確な応答を生成

#### 実装方法

```python
# RAGシステムとの統合
class RAGIntegratedVoiceService:
    def __init__(self):
        self.rag_engine = RAGEngine()  # 既存のRAGエンジン
        self.voice_service = VoiceResponseService()
    
    async def generate_response_with_rag(self, question: str):
        # 1. 音声認識
        text = await self.voice_service.transcribe_audio(audio)
        
        # 2. RAGで関連情報を検索
        context, sources = await self.rag_engine.query_with_rag(text)
        
        # 3. コンテキストを考慮した応答生成
        response = await self.generate_response(text, context)
        
        # 4. 音声合成
        audio = await self.voice_service.text_to_speech(response)
        
        return audio, response, sources
```

- **技術ドキュメント検索**: 「FastAPIの使い方を教えて」
- **コードベース検索**: 「このプロジェクトのアーキテクチャは？」

## 📊 実装例：RAG統合

### 実装コード

```python
# backend/services/rag_voice_service.py
from services.voice_response_service import VoiceResponseService
from services.rag_service import RAGService  # 既存のRAGサービス

class RAGVoiceService:
    """RAG統合音声応答サービス"""
    
    def __init__(self):
        self.voice_service = VoiceResponseService()
        self.rag_service = RAGService()
    
    async def process_with_rag(self, audio_chunk: bytes, connection_id: int):
        """RAGを活用した音声処理"""
        # 1. 音声認識
        text = await self.voice_service.transcribe_audio_chunk(
            audio_chunk, connection_id
        )
        
        if not text.get("text"):
            return None
        
        question = text["text"]
        
        # 2. RAGで関連情報を検索
        response, sources = await self.rag_service.query(
            question, top_k=3
        )
        
        # 3. 音声合成
        audio_response = await self.voice_service.text_to_speech(response)
        
        return {
            "text": response,
            "audio": audio_response,
            "sources": sources
        }
```
