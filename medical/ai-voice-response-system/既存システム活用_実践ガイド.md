# 既存システム活用 - 実践ガイド

## 🎯 概要

既存の面談準備資料を自動的に読み込んで、AI自動音声応答システムで活用する方法

## 🚀 クイックスタート

### ステップ1: 準備資料の読み込み

```bash
# APIエンドポイントを呼び出し
POST http://localhost:8001/api/interview/load-preparation/AIexe
```

### ステップ2: 練習用質問リストの取得

```bash
# 質問リストを取得
GET http://localhost:8001/api/interview/practice-questions
```

### ステップ3: 面談練習の開始

```bash
# 面談モードを有効化（WebSocket経由）
{
  "type": "set_interview_mode",
  "enabled": true
}
```

## 📋 詳細な使用方法

### 方法1: Pythonスクリプトで使用

```python
import requests

# 1. 準備資料を読み込む
response = requests.post("http://localhost:8001/api/interview/load-preparation/AIexe")
print(response.json())

# 2. 練習用質問リストを取得
response = requests.get("http://localhost:8001/api/interview/practice-questions")
questions = response.json()["questions"]

# 3. 各質問で練習
for question in questions[:5]:  # 最初の5問
    print(f"Q: {question}")
    # 音声で質問すると、AIが応答を生成
```

### 方法2: curlコマンドで使用

```bash
# 準備資料を読み込む
curl -X POST http://localhost:8001/api/interview/load-preparation/AIexe

# 質問リストを取得
curl http://localhost:8001/api/interview/practice-questions
```

### 方法3: ブラウザで使用

1. ブラウザで `http://localhost:8001` を開く
2. 開発者ツール（F12）を開く
3. Consoleで以下を実行：

```javascript
// 準備資料を読み込む
fetch('/api/interview/load-preparation/AIexe', {
    method: 'POST'
})
.then(res => res.json())
.then(data => console.log('読み込み完了:', data));

// 質問リストを取得
fetch('/api/interview/practice-questions')
.then(res => res.json())
.then(data => console.log('質問リスト:', data));
```

## 🔧 カスタマイズ方法

### 新しい企業の準備資料を追加

1. **企業情報ファイルを作成**
   ```
   government-ai-security-platform/{企業名}-面談準備ガイド.md
   ```

2. **質問事項リストを作成**
   ```
   government-ai-security-platform/{企業名}-質問事項リスト_面談用最終版.md
   ```

3. **統合サービスの`_load_company_info`メソッドを拡張**
   ```python
   def _load_company_info(self, company_name: str):
       if company_name == "新しい企業名":
           # 新しい企業の情報を読み込む
           ...
   ```
