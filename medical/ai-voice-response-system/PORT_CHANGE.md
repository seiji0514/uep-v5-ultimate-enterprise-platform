# ポート番号の変更方法

`http://localhost:8000` が別のシステムで使用中の場合、ポート番号を変更できます。

## 方法1: 環境変数で設定（推奨）

`.env`ファイルを作成または編集：

```env
SERVER_PORT=8001
```

## 方法2: 設定ファイルで変更

`config.py`ファイルを編集：

```python
SERVER_PORT = int(os.getenv("SERVER_PORT", "8001"))  # デフォルトを8001に変更
```

## 方法3: 起動時に環境変数で指定

```bash
# Windows
set SERVER_PORT=8001
py start.py

# Linux/Mac
export SERVER_PORT=8001
python start.py
```

## よく使用されるポート番号

- `8001` - よく使用される代替ポート
- `8080` - HTTPプロキシの代替ポート
- `3000` - Node.jsアプリケーションのデフォルト
- `5000` - Flaskアプリケーションのデフォルト
- `9000` - その他のWebアプリケーション

## 注意事項

1. **ポート番号を変更したら、ブラウザのURLも変更してください**
   - 例: `http://localhost:8001`

2. **WebSocket接続も自動的に新しいポートを使用します**
   - フロントエンドは自動的に現在のページのポートを使用します

3. **ファイアウォール設定**
   - 新しいポートがファイアウォールで許可されているか確認してください

## 確認方法

サーバー起動時に以下のメッセージが表示されます：

```
INFO - サーバーを起動します: http://0.0.0.0:8001
```

このポート番号でアクセスしてください。
