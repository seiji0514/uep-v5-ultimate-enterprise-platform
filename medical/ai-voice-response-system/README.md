# 🎤 AI自動音声応答システム

**リアルタイムで音声を認識し、AIが自動的に応答を生成して音声で返す完全自動の音声対話システム**

## ✨ 特徴

- 🎯 **完全自動**: 音声入力から音声出力まで自動処理
- 🚀 **リアルタイム**: WebSocketによる低遅延通信
- 💰 **無料対応**: ローカルモデルで完全無料で動作可能
- 🌐 **多言語対応**: 日本語・英語など100言語以上
- 🔧 **柔軟な設定**: 有料APIとローカルモデルの両方に対応

## 📋 機能

### 1. リアルタイム音声認識（ASR）
- マイクからの音声入力をリアルタイムで文字起こし
- **Whisper API**（高精度・高速）または**ローカルWhisper**（無料・オフライン）
- 日本語・英語など100言語以上に対応

### 2. AI応答生成
- 音声認識されたテキストをLLMで処理
- コンテキストを理解した自然な応答を生成
- **OpenAI GPT**（高品質）または**簡易応答システム**（無料）

### 3. 音声合成（TTS）
- 生成された応答テキストを音声に変換
- **pyttsx3**（ローカル・無料）または**transformers TTS**（高品質）
- 自然な音声で応答を返す

### 4. リアルタイム会話ループ
- 継続的な会話が可能
- 会話履歴の管理（最新10件）
- 音声の自動検出と処理

## 🛠️ 技術スタック

- **音声認識**: OpenAI Whisper / ローカルWhisper
- **応答生成**: OpenAI GPT-3.5/GPT-4 / 簡易応答システム
- **音声合成**: pyttsx3 / Microsoft SpeechT5
- **バックエンド**: FastAPI + WebSocket
- **フロントエンド**: HTML5 + Web Audio API
- **リアルタイム通信**: WebSocket

## 🚀 クイックスタート

### Pythonコマンドで起動（推奨）

```bash
# 1. プロジェクトフォルダに移動
cd ai-voice-response-system

# 2. Python Launcher (py) で起動
py start.py

# または python コマンドで起動
python start.py
```

### Windows（バッチファイル）

```bash
# 1. プロジェクトフォルダに移動
cd ai-voice-response-system

# 2. Python Launcher (py) を使用（推奨）
start_py.bat

# または通常のバッチファイル
start.bat
```

**注意**: `start.bat`は自動的に`py`コマンドまたは`python`コマンドを検出します。

### Linux/Mac（シェルスクリプト）

```bash
# 1. プロジェクトフォルダに移動
cd ai-voice-response-system

# 2. 実行権限を付与
chmod +x start.sh

# 3. 起動スクリプトを実行
./start.sh
```

### 手動起動

```bash
# 1. 依存関係のインストール
pip install -r requirements.txt

# 2. サーバー起動
cd backend
python main.py

# 3. ブラウザで http://localhost:8001 を開く
```

詳細なセットアップ手順は [INSTALL.md](INSTALL.md) を参照してください。

## 📖 使用方法

1. **サーバー起動**: `start.bat`（Windows）または `start.sh`（Linux/Mac）を実行
2. **ブラウザでアクセス**: `http://localhost:8001` を開く
3. **会話開始**: 「🎤 会話を開始」ボタンをクリック
4. **マイク許可**: ブラウザでマイクの使用を許可
5. **話す**: マイクに向かって話す
6. **応答確認**: AIが自動的に応答を音声で返します

## ⚙️ 設定（オプション）

`.env`ファイルを作成して設定をカスタマイズできます：

```env
# OpenAI API Key（オプション）
# 設定すると、より高精度な音声認識と応答生成が可能です
OPENAI_API_KEY=your_api_key_here

# サーバー設定
SERVER_HOST=0.0.0.0
SERVER_PORT=8001

# Whisperモデル（tiny, base, small, medium, large）
WHISPER_MODEL=base

# AIモデル
AI_MODEL=gpt-3.5-turbo

# TTSエンジン（pyttsx3, transformers）
TTS_ENGINE=pyttsx3
```

**ローカルモデルのみ使用する場合**: `.env`ファイルは不要です。
