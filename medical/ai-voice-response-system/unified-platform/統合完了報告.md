## 🚀 使用方法

### 1. 起動

```bash
cd unified-platform
start.bat
```

### 2. アクセス

ブラウザで `http://localhost:8001` にアクセス

### 3. 分野選択と会話

1. 分野を選択（医療、法務、金融など）
2. 「会話を開始」ボタンをクリック
3. マイクに向かって話す
4. AIが分野特化の応答を返す
