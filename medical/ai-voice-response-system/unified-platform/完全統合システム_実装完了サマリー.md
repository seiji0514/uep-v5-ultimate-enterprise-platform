## 🚀 使用方法

### 1. インストール

```bash
cd unified-platform
pip install -r requirements.txt
```

### 2. 環境変数の設定

`.env`ファイルを作成：

```env
SERVER_HOST=0.0.0.0
SERVER_PORT=8001
OPENAI_API_KEY=your_api_key_here
```

### 3. 起動

```bash
start.bat  # Windows
# または
cd backend
python main.py
```

### 4. アクセス

ブラウザで `http://localhost:8001` にアクセス

## 🔧 既存システムとの統合

各モジュールの`_initialize_*_service()`メソッドで、既存システムのパスを設定してください。

### 例: 医療システム統合

```python
# backend/modules/healthcare_module.py
def _initialize_healthcare_service(self):
    healthcare_path = PROJECT_ROOT / "次世代マルチモーダルAI統合プラットフォームv8.0" / "app" / "services" / "healthcare_ai.py"
    if healthcare_path.exists():
        # 既存システムをインポート
        from app.services.healthcare_ai import HealthcareAIService
        self.healthcare_service = HealthcareAIService()
```
