## 🚀 次のステップ

### 1. 動作確認

```bash
cd unified-platform
start.bat
```

**確認ポイント**:
- [ ] ポートが正常にバインドされる（エラーが出ない）
- [ ] 既存システムが見つかる（警告が出ない、または警告が出ても動作する）
- [ ] ブラウザで `http://localhost:8001`（または表示されたポート）にアクセスできる

### 2. ログの確認

起動時のログで以下を確認：

**成功パターン**:
```
医療サービスが見つかりました: ...
医療サービスを初期化しました
完全統合AI音声応答プラットフォームを起動します: http://0.0.0.0:8001
```

**フォールバックパターン（動作は可能）**:
```
医療サービスが見つかりませんでした。フォールバックモードで動作します。
完全統合AI音声応答プラットフォームを起動します: http://0.0.0.0:8002
```

### 3. 問題が続く場合

1. **デバッグログを有効化**:
   ```python
   # backend/config.py
   LOG_LEVEL = "DEBUG"
   ```

2. **パスを手動で確認**:
   - ログに表示される`PROJECT_ROOT`を確認
   - 既存システムのパスが正しいか確認

3. **トラブルシューティングガイドを参照**:
   - `トラブルシューティング.md`を確認
