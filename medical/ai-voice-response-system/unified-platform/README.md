# 完全統合AI音声応答プラットフォーム

## 🎯 概要

AI自動音声応答システムを含む、様々な分野に対応した完全統合システム

## ✨ 特徴

- 🎯 **完全統合**: 複数分野を1つのプラットフォームで
- 🚀 **音声インターフェース**: すべての分野で音声で操作
- 🔧 **既存システム統合**: 医療、法務、金融システムと統合
- 📊 **分野別最適化**: 各分野に特化した応答生成

## 📋 対応分野

### 完全統合（既存システムと統合）

1. **医療・ヘルスケア**
   - 既存システム: `healthcare_ai.py`
   - 機能: 症状相談、健康管理、医療情報検索

2. **法務・法律**
   - 既存システム: `contract-review-system`
   - 機能: 契約書分析、法務Q&A、法令検索

3. **金融・FinTech**
   - 既存システム: `fintech_ai.py`
   - 機能: 不正検知、リスク評価、金融相談

### 基本実装（拡張可能）

4. **教育・学習** - 学習支援、クイズ生成
5. **製造業** - 品質管理、異常検知
6. **小売・EC** - 商品検索、レコメンデーション

## 🚀 クイックスタート

### セットアップ

```bash
cd unified-platform
pip install -r requirements.txt
```

### 起動

```bash
cd backend
python main.py
```

### アクセス

```
http://localhost:8001
```

## 📖 使用方法

1. **ブラウザでアクセス**: `http://localhost:8001`
2. **分野を選択**: 医療、法務、金融など
3. **会話開始**: 「会話を開始」ボタンをクリック
4. **音声で質問**: マイクに向かって話す
5. **応答確認**: AIが分野特化の応答を音声で返す

## 🔧 分野の切り替え

### WebSocket経由

```javascript
// 医療分野に切り替え
ws.send(JSON.stringify({
    type: "set_domain",
    domain: "healthcare"
}));

// 法務分野に切り替え
ws.send(JSON.stringify({
    type: "set_domain",
    domain: "legal"
}));
```

```bash

```
