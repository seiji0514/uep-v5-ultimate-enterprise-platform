# 外部システム統合 - 実践ガイド

## 🎯 概要

AI自動音声応答システムを外部システムと統合して、より多機能な音声アシスタントとして活用する方法

## 🚀 実装済み機能

### 1. 意図検出システム

音声入力からユーザーの意図を自動検出：

- **天気**: 「今日の天気は？」
- **カレンダー**: 「今日の予定を教えて」
- **タスク**: 「タスクを作成して」
- **メール**: 「メールを確認して」
- **ファイル**: 「ファイルを検索して」
- **RAG検索**: 「調べて」「教えて」

### 2. 外部API連携

以下の外部APIと連携可能：

- 天気情報API
- カレンダーAPI
- タスク管理API
- メールAPI
- ファイル管理API

## 📋 使用方法

### 方法1: 外部APIを追加

```bash
POST http://localhost:8001/api/external/add-api
Content-Type: application/json

{
  "name": "weather",
  "api_config": {
    "base_url": "https://api.openweathermap.org",
    "api_key": "your_api_key",
    "endpoints": {
      "current": "/data/2.5/weather"
    }
  }
}
```

### 方法2: RAGサービスと統合

```bash
POST http://localhost:8001/api/external/integrate-rag
Content-Type: application/json

{
  "rag_service_path": "llm-rag-chatbot/rag_engine.py",
  "config": {
    "vectorstore_path": "./data/vectorstore",
    "embedding_model": "sentence-transformers/all-MiniLM-L6-v2"
  }
}
```

## 🎯 実用的な使用例

### 例1: 天気情報の取得

```
ユーザー: 「今日の天気は？」
↓
システム: 意図を検出（weather）
↓
システム: 天気APIを呼び出し
↓
応答: 「今日の天気は晴れ、気温は25度です」
（音声で応答）
```

### 例2: タスク管理

```
ユーザー: 「タスクを作成して、明日の会議の準備」
↓
システム: 意図を検出（task, action=create）
↓
システム: タスク管理APIを呼び出し
↓
応答: 「タスク「明日の会議の準備」を作成しました」
（音声で応答）
```

### 例3: RAG検索

```
ユーザー: 「FastAPIの使い方を調べて」
↓
システム: 意図を検出（rag）
↓
システム: RAGエンジンで検索
↓
応答: 「FastAPIはPythonのWebフレームワークで...」
（音声で応答 + 関連ドキュメントを表示）
```

## 🔧 カスタマイズ方法

### 新しい外部APIを追加

```python
# backend/services/external_integration_service.py を編集

async def _handle_custom_api(self, text: str, entities: Dict) -> str:
    """カスタムAPIを処理"""
    # カスタムAPIを呼び出し
    response = await call_custom_api(text)
    return response
```

### 新しい意図を追加

```python
# 意図検出に新しいキーワードを追加
intents = {
    "custom": ["カスタムキーワード1", "カスタムキーワード2"],
    # ...
}
```

## 📝 まとめ

既存システム以外でも、以下の方法でAI自動音声応答システムを活用できます：

1. **外部API連携** - リアルタイム情報取得
2. **RAGシステム統合** - 知識ベース検索
3. **データベース連携** - データ検索・分析
4. **タスク管理連携** - タスク操作
5. **IoTデバイス連携** - スマートホーム制御

これらを組み合わせることで、**汎用的な音声アシスタントシステム**として活用できます。
