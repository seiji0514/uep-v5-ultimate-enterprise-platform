# 医療データ異常検知MLOpsプラットフォーム - プロトタイプ

## 概要
てんかん脳波データMLOps環境を参考にした、医療データ異常検知システムのプロトタイプ

## 実装時間
- **総実装時間**: 2-3時間
- **簡易版**: 既存技術を最大限活用

## 技術スタック
- Python 3.11
- scikit-learn（異常検知）
- Pandas/NumPy（データ処理）
- FastAPI（API Server）
- HTML/JavaScript（ダッシュボード）

## システム構成

```
医療データ異常検知MLOpsプラットフォーム
├── データ収集・前処理
│   ├── CSVファイル読み込み
│   ├── 基本的なフィルタリング
│   └── 特徴抽出
├── 異常検知モデル
│   ├── Isolation Forest
│   └── 統計的手法
├── MLOps基盤
│   ├── モデル保存・ロード
│   ├── バージョン管理
│   └── 評価指標記録
└── ダッシュボード
    ├── リアルタイム監視
    ├── 異常アラート
    └── 精度メトリクス
```

## 起動方法

### 1. 依存パッケージのインストール
```bash
pip install -r requirements.txt
```

### 2. サンプルデータ生成
```bash
python generate_sample_data.py
```

### 3. 異常検知モデルの学習
```bash
python train_model.py
```

### 4. APIサーバーの起動
```bash
uvicorn api_server:app --reload --port 8000
```

### 5. ダッシュボードを開く
```bash
start medical_dashboard.html
```
