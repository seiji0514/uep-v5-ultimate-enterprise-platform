## 4. 完了しない場合の確認ポイント

| 確認項目           | 内容                                     |
| ------------------ | ---------------------------------------- |
| **学歴の必須項目** | 学校名は必ず入力（空欄だと保存できない） |
| **保存ボタン**     | 各セクションで「保存する」をクリック     |
| **ブラウザ**       | キャッシュ削除・再読み込みを試す         |
| **入力内容**       | 文字数制限や特殊文字の有無を確認         |

---

## 5. 自己紹介・学歴・その他の経験の入力順

1. **自己紹介** → 保存
2. **学歴** → 学校名必須、保存
3. **その他の経験** → 保存

---

## 6. 学歴が不明な場合

- 高卒・専門卒・大卒など、**最終学歴**を1件登録すればOK
- 学校名・卒業年が不明な場合は、証明書や履歴書で確認
- 中退の場合は「卒業状況」で「中退」など適切な項目を選択
