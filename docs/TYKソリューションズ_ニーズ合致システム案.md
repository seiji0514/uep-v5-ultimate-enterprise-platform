# TYKソリューションズ ニーズ合致システム案

TYKの事業・案件ニーズに合うUEPの既存システムと、提案可能なシステム案です。

---

## TYKのニーズ（求人・特色より）

| ニーズ | 内容 |
|--------|------|
| AI技術の社会実装 | 業務効率化・AIの実社会での活用 |
| AI搭載スマートグラス | ウェアラブル・現場向けAI |
| ローカルAI | 端末内・オフラインでのAI推論 |
| AIエージェント開発 | タスク自動化・エージェント連携 |
| システム開発・ITインフラ | プラットフォーム構築・基盤 |

---

## 既存システムでニーズに合うもの

| TYKのニーズ | UEPの既存システム | 対応 |
|-------------|-------------------|------|
| **AIエージェント** | インクルーシブ雇用AI（マッチング・相談・評価エージェントのオーケストレーション） | デモ可能 |
| **AI技術の社会実装** | 医療・製造・金融・障害者雇用・契約・物流の6モジュール統合基盤 | デモ可能 |
| **業務効率化** | 契約ワークフロー（見積・契約・納品・請求を一気通貫で管理） | デモ可能 |
| **AIプラットフォーム** | UEP v5.0 全体（MLOps・生成AI・RAG・CoT推論） | デモ可能 |
| **システム開発・インフラ** | Docker・K8s・Terraform・CI/CD・監視基盤 | 実装済み |

---

## 提案可能なシステム案（新規・拡張）

| 案 | 内容 | TYKニーズとの接点 |
|----|------|-------------------|
| **スマートグラス向けUI** | インクルーシブ雇用AI（障害者雇用マッチング・アクセシビリティ特化AI）を、スマートグラス向けの軽量UI・音声操作で提供 | スマートグラス・ローカルAI |
| **現場AIエージェント** | 製造・医療・物流の現場向けに、タスク確認・異常検知・音声指示を端末内で実行するAIエージェント | ローカルAI・AIエージェント |
| **業務効率化PoC** | 契約・納品・請求のワークフローを、顧客の既存システムとAPI連携してPoC | 業務効率化・社会実装 |
| **障害者雇用×AI** | 障害者雇用マッチング・アクセシビリティ評価を、企業の採用サイトに組み込むSaaS | 社会実装・業務効率化 |

## 面談での伝え方の例

1. **既存デモで伝える**  
   「6モジュール統合基盤とインクルーシブ雇用AIをデモでご覧いただけます。AIエージェントのオーケストレーションや、製造・医療・金融・物流の業務効率化は実装済みです。」

2. **ニーズを聞いてから提案**  
   「貴社のスマートグラス・ローカルAI・AIエージェントの案件に、インクルーシブ雇用AIをスマートグラス向けUIに拡張する形で貢献できます。」

3. **逆質問で確認**  
   「貴社のAIエージェント開発やスマートグラス案件で、どのような技術・ユースケースを想定されていますか？」

- **既存デモ**: 6モジュール・インクルーシブ雇用AI・AIエージェント基盤で、TYKのニーズに合わせて説明可能
- **提案案**: スマートグラス向けUI、現場AIエージェント、業務効率化PoC など、ニーズに応じて提案可能
