# セットアップ手順（拡張機能）

**作成日**: 2026年3月  
**対象**: uep-cli、uep-operator、オプション依存パッケージ

---

## 前提条件

- **Go**: 1.19以上（uep-cli、uep-operator 用）
- **Node.js**: 18以上（フロントエンド用）
- **Python**: 3.11以上（バックエンド用）

---

## 1. フロントエンド

```cmd
cd frontend
npm install
```

**Windows（PowerShell）**:
```powershell
cd frontend; npm install
```

---

## 2. uep-cli（Python版・推奨）

**Go exe は McAfee 等のセキュリティソフトで隔離されるため、Python版を推奨します。**

```cmd
cd tools\uep-cli
python uep-cli.py version
python uep-cli.py health
python uep-cli.py events list
```

**バッチで実行**:
```cmd
tools\uep-cli\run-uep-cli.bat version
tools\uep-cli\run-uep-cli.bat health
tools\uep-cli\run-uep-cli.bat events list
```

**Go exe 版**（セキュリティソフトの除外設定ができる場合）:
```cmd
cd tools\uep-cli
go build -o uep-cli.exe .
uep-cli.exe health
```

---

## 3. uep-operator（Kubernetes Operator）

```cmd
cd tools\uep-operator
go build -o uep-operator.exe .
```

**Windows（PowerShell）**:
```powershell
cd tools\uep-operator
go build -o uep-operator.exe .
```

**実行例**:
```cmd
uep-operator.exe
```

※Kubernetes クラスターが利用可能な環境で実行してください。

---

## 4. オプション依存パッケージ（Python）

バックエンドの機能拡張で、特定の機能を使う場合にのみインストールします。

### バックエンドディレクトリへ移動

```cmd
cd backend
```

### 個別インストール

| 機能 | コマンド | 用途 |
|------|----------|------|
| **Redis** | 既に requirements.txt に含まれています | キャッシュ、Redis Streams |
| **fhirclient** | `pip install fhirclient` | 医療 FHIR 連携 |
| **opcua** | `pip install opcua` | 製造 OPC-UA 連携 |
| **langgraph** | `pip install langgraph langchain` | LangGraph エージェント |

### 一括インストール（オプション機能を全て使う場合）

```cmd
pip install fhirclient opcua langgraph langchain
```

---

## 5. 実行順序まとめ

```cmd
REM 1. フロントエンド
cd frontend
npm install
cd ..

REM 2. uep-cli
cd tools\uep-cli
go build -o uep-cli.exe .
cd ..\..

REM 3. uep-operator
cd tools\uep-operator
go build -o uep-operator.exe .
cd ..\..

REM 4. オプション依存（必要な場合のみ）
cd backend
pip install fhirclient opcua langgraph langchain
```

## 6. トラブルシューティング

| 現象 | 対処 |
|------|------|
| `go: command not found` | Go をインストール: https://go.dev/dl/ |
| `npm: command not found` | Node.js をインストール: https://nodejs.org/ |
| `pip install opcua` でエラー | `opcua` は `asyncua` が推奨の場合あり。`pip install asyncua` を試す |
| FHIR エンドポイント 404 | FHIR サーバー（例: HAPI FHIR）を別途起動 |
| `uep-cli.exe` が実行できない | `tools\uep-cli\run-uep-cli.bat version` で実行。または Windows Defender の除外設定を確認 |
| uep-operator ビルド失敗 | `cd tools\uep-operator` で `go mod tidy` を実行してから `go build` |
