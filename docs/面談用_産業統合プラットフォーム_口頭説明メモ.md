# 面談用：産業統合プラットフォーム シミュレーション 口頭説明メモ

**デモ**: UEP v5.0 を操作  
**口頭説明**: 産業統合プラットフォーム シミュレーションシステムの概要

---

## 1. 冒頭（30秒）

「UEP v5.0 とは別に、**産業統合プラットフォーム**というシミュレーションシステムも構築しています。製造・医療・金融・セキュリティなど、複数業種を1つの画面で統合したデモです。概要をご説明します。」

---

## 2. システム概要（1分）

### 何をするシステムか

- **8つのドメイン**を1つの画面でシミュレーション
- 要対応・要確認を横断的に把握できるダッシュボード
- 業種・プロジェクトに応じて重点ドメインを切り替えて表示

### 実務学習（全8ドメイン共通）

- **順番通りに進めれば実務が学べる**。各ドメインの「実務学習」チップで、順番に操作して習得
- ステップ形式で選択肢を選び、正解なら次へ。全ステップ完了で習得完了

| ドメイン                     | 実務学習の内容（製造・IoTと同様に分離・詳細説明付き）                                          |
| ---------------------------- | ---------------------------------------------------------------------------------------------- |
| 製造・IoT                    | 予知保全／センサーデータ（OPC-UA）／異常検知 の3種。各2〜3ステップ、正解・不正解の理由説明あり |
| 医療・ヘルスケア             | AI診断／音声応答／異常検知 の3種。各2ステップ                                                  |
| 金融・FinTech                | 決済／リスクスコア／取引監視／ストレステスト の4種。各2ステップ                                |
| 統合セキュリティ・防衛       | イベント／インシデント／リスク の3種。各2ステップ                                              |
| インフラ・運用系             | IaC／オーケストレーション／監視／最適化 の4種。各2ステップ                                     |
| DX・データ系                 | データレイク／MLモデル／生成AI／パイプライン の4種。各2ステップ                                |
| コンプライアンス・ガバナンス | 規制対応／監査ログ／ポリシー／監査指摘 の4種。各2ステップ                                      |
| サプライチェーン             | 物流／在庫／調達／需要予測 の4種。各2ステップ                                                  |

## 3. 主な機能（1分）

### 要対応サマリ

- 全ドメインの要対応・要確認件数を一覧表示
- チップをクリックすると該当タブに遷移
- 製造の要メンテ、医療の要確認、金融の高リスク、セキュリティのCritical/High などを集約

### シミュレーションモード

- **自動モード**: 5秒・10秒・30秒間隔でデータを自動再取得
- **手動モード**: 「全データ更新」で任意のタイミングで更新
- **順番ガイド**: タブを順番に自動切り替え（製造→医療→金融→…）。確認すべきドメインの順番の目印。5/10/15/30秒間隔で選択可能

### 業種別対応

- 製造業 → 製造・IoT、ERP が中心
- 金融機関 → 金融・FinTech、セキュリティが中心
- 病院・医療法人 → 医療・ヘルスケアが中心
- SIer・クラウドベンダー → インフラ・運用系が中心

---

## 4. 技術的なポイント（30秒）

- バックエンド: FastAPI、ポート9010
- フロントエンド: React、MUI、ポート3010
- 製造（OPC-UA想定）、医療（FHIR想定）、金融（決済・リスク）など業種別APIを統合
- 各ドメインに豊富なサンプルデータ（8～10件程度）を用意。デモデータを実データに置き換えれば、PoC・検証に活用可能

---

## 5. 締め（20秒）

「産業統合プラットフォームは、複数業種を横断した統合基盤のイメージを、1つの画面でシミュレーションできるシステムです。必要であれば、別途デモの起動も可能です。」

---

## 6. コピペ用（一言でまとめ）

**産業統合プラットフォーム**

UEP v5.0とは別に、産業統合プラットフォームというシミュレーションも構築しています。製造・医療・金融・セキュリティなど8ドメインを1画面で統合。要対応を横断把握できる構成です。必要であれば別途デモも可能です。

**補足**: 個人会計システムもあり、経費管理・領収書OCR・請求書出力などを実装しています。
