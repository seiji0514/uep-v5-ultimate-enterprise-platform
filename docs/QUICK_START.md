# UEP v5.0 - クイックスタートガイド

**作成日**: 2026年1月29日

---

## 🚀 最も簡単な起動方法

### Windows（コマンドプロンプト）

```cmd
start-all.bat
```

これだけで、バックエンドとフロントエンドが起動します！

---

## 📋 起動後のアクセス

起動が完了すると、以下のURLにアクセスできます：

- **バックエンドAPI**: http://localhost:8000
- **APIドキュメント**: http://localhost:8000/docs
- **フロントエンド**: http://localhost:3000（存在する場合）

---

## 🛑 停止方法

### 方法1: スクリプトで停止（推奨）

```cmd
stop-all.bat
```

### 方法2: 手動で停止

各ウィンドウで **`Ctrl+C`** を押すか、**ウィンドウを閉じる**

---

## 📝 個別起動

### バックエンドのみ

```cmd
start-backend.bat
```

### フロントエンドのみ

```cmd
start-frontend.bat
```

---

## ⚠️ 注意事項

### フロントエンドが存在しない場合

`start-frontend.bat`を実行すると、フロントエンドの作成を促されます。

- **Y** を入力: React + TypeScriptプロジェクトが自動的に作成されます
- **N** を入力: フロントエンドの起動をスキップします

### 初回起動時

- 仮想環境が自動的に作成されます
- 依存パッケージが自動的にインストールされます
- `.env`ファイルが自動的に作成されます

---

## 🔧 前提条件

### バックエンド

- ✅ Python 3.11以上
- ✅ pip（Pythonに含まれています）

### フロントエンド（オプション）

- ✅ Node.js 18以上
- ✅ npm（Node.jsに含まれています）
