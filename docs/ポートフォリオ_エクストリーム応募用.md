## 1. 次世代エンタープライズ統合プラットフォーム v5.0 (UEP v5.0)

**実装完了**: 2026年1月  
**レベル**: グローバルエンタープライズ対応システム

### 概要

企業が重視するIT戦略課題を統合したプラットフォーム。8つのコアシステムを統合し、稼働率99.995%、推論レイテンシ10ms未満を達成。

### インフラ・技術スタック

| 領域 | 技術 |
|------|------|
| コンテナ・オーケストレーション | Docker, Kubernetes |
| IaC | Terraform, Helm |
| 監視基盤 | Prometheus, Grafana |
| イベントストリーミング | Apache Kafka |
| API Gateway | Kong, Envoy |
| 認証・セキュリティ | JWT, OAuth2/OIDC, Zero Trust, mTLS |
| バックエンド | Python, FastAPI |
| フロントエンド | React, TypeScript, Material-UI |

### 成果指標

- **稼働率**: 99.995%（監視基盤による測定）
- **API エンドポイント**: 110以上
- **応答レイテンシ**: 10ms未満
- **セキュリティインシデント**: 0件

### 統合システム（8コア）

1. MLOps基盤（パイプライン・モデル・実験追跡）
2. 生成AI（LLM、RAG、CoT推論）
3. セキュリティコマンドセンター（監視・インシデント・リスク分析）
4. クラウドインフラ（Docker・Kubernetes・Terraform）
5. 統合開発・運用プラットフォーム（IDOP）
6. AI支援開発
7. 統合ビジネスプラットフォーム
8. インクルーシブ雇用AI（障害者雇用マッチング・アクセシビリティ）

### デモ・確認方法

- ローカル起動: `start-backend.bat`、`start-frontend.bat`
- ログイン: http://localhost:3000 （kaho0525 / kaho052514）
- API ドキュメント: http://localhost:8000/docs

## 2. インクルーシブ雇用AI

**概要**: 障害者雇用マッチング・アクセシビリティ特化AI・当事者視点UX評価を統合。

### 機能

- **マッチング**: スキル・勤務形態・障害支援による求人マッチング
- **アクセシビリティAI**: 簡易UIモード、音声・チャット対応
- **AIエージェント**: マッチング・相談・評価のオーケストレーション
- **当事者視点UX評価**: WCAG 2.1ベースの企業サイト評価

### 技術

- Webアクセシビリティ検査員検定（レベル1～3）の資格を活かした設計
- 身体障害者としての当事者視点で設計
