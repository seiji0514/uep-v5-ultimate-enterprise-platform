## ITエンジニア実務経験

| 選択肢             | 内容 | 推奨       |
| ------------------ | ---- | ---------- |
| 1\_なし            | -    |            |
| 2_1年未満          | -    | ✓ **推奨** |
| 3_1年以上～2年未満 | -    |            |
| 4_2年以上～3年未満 | -    |            |

**理由**: KDDIエボルバ（ヘルプデスク・サーバー処理）11ヶ月＋AI技術研究開発8ヶ月。企業勤務でのIT関連経験を合わせて約1年未満～1年程度。フォームの選択肢が「2年以上～3年未満」までなので、**2_1年未満**が妥当。個人開発8ヶ月を実務に含めるかは解釈によるが、保守的に2_1年未満を推奨。

## 記入時の注意

1. **職務経歴**: 最大2000文字（事務局確認済み）。**type** と **type エンジニア転職フェア ONLINE** は別サービス。本フォーム用に850文字追加可能なため、可能な限り詳細を記載した記入案（約1980文字）を推奨。
2. **フルリモート**・**身体障害者等**の必須情報も職務経歴欄に記載可能（事務局確認済み）。
3. **所属先**: 非公開可。会社名を伏せたい場合は「某社」「A社」等で可。
4. **ITエンジニア実務経験**: 企業勤務のIT関連経験年数で判断。不明な場合は2_1年未満を推奨。
5. **経験業種・職種・スキル**: 当てはまるものは複数選択。

## 戦略メモ

### 職務経歴書の使い分け

| 種類                                    | 用途                                 | 形式・制約                                                                                     | 容量・分量    | ファイル例                                      |
| --------------------------------------- | ------------------------------------ | ---------------------------------------------------------------------------------------------- | ------------- | ----------------------------------------------- |
| **通常の職務経歴書**                    | 企業への応募・送付（履歴書とセット） | 自由形式、企業ごとにカスタマイズ可、詳細に記載可能                                             | 約9ページ程度 | 職務経歴書*ラスク*開発エンジニア.md 等          |
| **type エンジニア転職フェア ONLINE 用** | フェア参加登録フォームの職務経歴欄   | 2000文字以内、【サマリ】【経験スキル】等のフォーマットに沿う、フルリモート・身体障害者等を含む | 約2ページ相当 | 職務経歴書\_typeエンジニア転職フェアONLINE用.md |

**推奨**：両方を作成・運用する。フェア用は職務経歴書\_typeエンジニア転職フェアONLINE用.md を参照。通常用（約9ページ）は企業ごとに志望動機・強調ポイントを調整して送付。フェアで面談が決まった企業には、通常の職務経歴書を別途送付することを推奨。2月25日までに type のプロフィールも同内容で更新すると、相乗効果が期待できる。
