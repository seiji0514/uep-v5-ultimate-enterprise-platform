### 通常の職務経歴書との容量差

| 種類                                                            | 容量・分量                                                                                            |
| --------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| **通常の職務経歴書**（例：職務経歴書*ラスク*開発エンジニア.md） | 約9ページ程度。詳細な職務内容・プロジェクト・技術スタック・自己PR等を記載。                           |
| **本ドキュメント**（type エンジニア転職フェア ONLINE 用）       | 職務経歴欄 最大2000文字。フォーム制約のため要約形式。【サマリ】【職務経歴概要】【経験スキル】に集約。 |

※フェアで面談が決まった企業には、通常の職務経歴書（9ページ程度）を別途送付することを推奨。
