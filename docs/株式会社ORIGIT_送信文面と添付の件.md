## PDF化の手順

### 方法1: MarkdownをWord等に貼り付けてPDF化

1. 各 `.md` ファイルを開き、内容をコピー
2. Word / Googleドキュメント等に貼り付け、体裁を整える
3. 「名前を付けて保存」→ PDF形式で保存

### 方法2: VS Code / Cursor の Markdown PDF 拡張機能

1. Markdown PDF 拡張機能をインストール
2. `.md` ファイルを開き、右クリック → 「Markdown PDF: Export (pdf)」を選択
3. 同フォルダにPDFが生成される

### 方法3: オンラインツール

- [Pandoc](https://pandoc.org/) で `md` → `pdf` 変換
- [Dillinger](https://dillinger.io/) 等のMarkdownエディタでPDFエクスポート
