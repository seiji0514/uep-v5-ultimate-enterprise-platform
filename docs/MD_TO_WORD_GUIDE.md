# MarkdownからWordへの移行ガイド

**作成日**: 2026年2月11日  
**目的**: 職務経歴書（Markdown）をWordに移行する際の方法と注意点

---

## 📋 推奨方法

### 方法1: Wordの「形式を保持して貼り付け」を使用（推奨）

**手順**:

1. **Markdownファイルを開く**
   - `docs/RESUME_REVISED.md` を開く
   - 全体を選択（Ctrl+A）してコピー（Ctrl+C）

2. **Wordを開く**
   - 新しいWord文書を作成

3. **形式を保持して貼り付け**
   - Wordで「ホーム」タブ → 「貼り付け」の下矢印 → 「形式を保持して貼り付け」を選択
   - または、`Ctrl+Shift+V`（Word 2016以降）

4. **手動でフォーマット調整**
   - 見出し（#、##、###）をWordの見出しスタイルに変換
   - 太字（**）が正しく表示されているか確認
   - リスト（-）が正しく表示されているか確認
   - 区切り線（---）をWordの区切り線に変換

**メリット**:
- ✅ 基本的なフォーマットが保持される
- ✅ 太字やリストが正しく表示されることが多い

**デメリット**:
- ⚠️ 見出しスタイルは手動で設定する必要がある
- ⚠️ 区切り線は手動で設定する必要がある

---

### 方法2: 書式なしで貼り付け + 手動フォーマット（最も確実）

**手順**:

1. **Markdownファイルを開く**
   - `docs/RESUME_REVISED.md` を開く
   - 全体を選択（Ctrl+A）してコピー（Ctrl+C）

2. **Wordを開く**
   - 新しいWord文書を作成

3. **書式なしで貼り付け**
   - Wordで「ホーム」タブ → 「貼り付け」の下矢印 → 「テキストのみ保持」を選択
   - または、`Ctrl+Alt+V` → 「テキスト（Unicode）」を選択

4. **手動でフォーマット設定**
   - **見出し1（#）**: 「ホーム」タブ → 「見出し1」スタイルを適用
   - **見出し2（##）**: 「ホーム」タブ → 「見出し2」スタイルを適用
   - **見出し3（###）**: 「ホーム」タブ → 「見出し3」スタイルを適用
   - **太字（**）**: 該当テキストを選択 → `Ctrl+B` または「ホーム」タブ → 「太字」
   - **リスト（-）**: 該当行を選択 → 「ホーム」タブ → 「箇条書き」
   - **区切り線（---）**: 「挿入」タブ → 「図形」 → 「線」を選択

**メリット**:
- ✅ 確実にフォーマットを制御できる
- ✅ 見出しスタイルが正しく設定される
- ✅ プロフェッショナルな見た目になる

**デメリット**:
- ⚠️ 手動での作業が必要
- ⚠️ 時間がかかる

---

### 方法3: Pandocを使用（技術者向け）

**手順**:

1. **Pandocをインストール**
   - https://pandoc.org/installing.html からインストール

2. **コマンドを実行**
   ```bash
   pandoc docs/RESUME_REVISED.md -o RESUME_REVISED.docx
   ```

3. **Wordで開く**
   - 生成された `RESUME_REVISED.docx` をWordで開く

**メリット**:
- ✅ 自動的にMarkdownをWordに変換
- ✅ 見出しスタイルが正しく設定される
- ✅ リストや太字が正しく表示される

**デメリット**:
- ⚠️ Pandocのインストールが必要
- ⚠️ コマンドライン操作が必要

## ⚠️ 注意点

### 1. Markdown記法がそのまま表示される可能性

**問題**:
- `# 見出し` がそのまま表示される
- `**太字**` がそのまま表示される
- `---` がそのまま表示される

**解決方法**:
- 「書式なしで貼り付け」を使用
- または、手動でMarkdown記法を削除

---

### 2. 見出しスタイルの設定

**問題**:
- `# 職務経歴書（修正版）` が通常のテキストとして表示される

**解決方法**:
- 該当テキストを選択
- 「ホーム」タブ → 「見出し1」スタイルを適用
- または、`Ctrl+Alt+1`（見出し1）、`Ctrl+Alt+2`（見出し2）、`Ctrl+Alt+3`（見出し3）

---

### 3. リストの設定

**問題**:
- `- リスト項目` が通常のテキストとして表示される

**解決方法**:
- 該当行を選択
- 「ホーム」タブ → 「箇条書き」を選択
- または、`Ctrl+Shift+L`

---

### 4. 区切り線の設定

**問題**:
- `---` がそのまま表示される

**解決方法**:
- `---` を削除
- 「挿入」タブ → 「図形」 → 「線」を選択
- または、「ホーム」タブ → 「罫線」 → 「横罫線」

---

### 5. 太字の設定

**問題**:
- `**太字**` がそのまま表示される

**解決方法**:
- `**` を削除
- 該当テキストを選択
- `Ctrl+B` または「ホーム」タブ → 「太字」

---

### 6. 改行の設定

**問題**:
- Markdownの改行（2スペース+改行）が正しく表示されない

**解決方法**:
- 必要に応じて手動で改行を調整
- 段落の間隔を調整（「ホーム」タブ → 「段落」 → 「間隔」）

## 📋 推奨手順（詳細版）

### ステップ1: Markdownファイルを開く

1. `docs/RESUME_REVISED.md` を開く
2. 全体を選択（`Ctrl+A`）
3. コピー（`Ctrl+C`）

### ステップ2: Wordに貼り付け

1. 新しいWord文書を作成
2. 「ホーム」タブ → 「貼り付け」の下矢印 → 「テキストのみ保持」を選択
   - または、`Ctrl+Alt+V` → 「テキスト（Unicode）」を選択

### ステップ3: 見出しスタイルを設定

1. **見出し1（#）**: `# 職務経歴書（修正版）`
   - 該当テキストを選択 → `Ctrl+Alt+1` または「ホーム」タブ → 「見出し1」

2. **見出し2（##）**: `## 【基本情報】`、`## 【資格】` など
   - 該当テキストを選択 → `Ctrl+Alt+2` または「ホーム」タブ → 「見出し2」

3. **見出し3（###）**: `### 1. AI技術研究開発（2025年6月～現在）` など
   - 該当テキストを選択 → `Ctrl+Alt+3` または「ホーム」タブ → 「見出し3」

### ステップ4: 太字を設定

1. `**作成日**`、`**氏名**` などの `**` を削除
2. 該当テキストを選択 → `Ctrl+B` または「ホーム」タブ → 「太字」

### ステップ5: リストを設定

1. `- **Webアクセシビリティ検査員検定**` などの `-` を削除
2. 該当行を選択 → 「ホーム」タブ → 「箇条書き」
   - または、`Ctrl+Shift+L`

### ステップ6: 区切り線を設定

1. `---` を削除
2. 「挿入」タブ → 「図形」 → 「線」を選択
3. または、「ホーム」タブ → 「罫線」 → 「横罫線」

### ステップ7: 最終確認

1. **見出しスタイル**: すべての見出しが正しく設定されているか確認
2. **太字**: すべての太字が正しく設定されているか確認
3. **リスト**: すべてのリストが正しく設定されているか確認
4. **区切り線**: すべての区切り線が正しく設定されているか確認
5. **改行**: 段落の間隔が適切か確認
6. **ページ番号**: 必要に応じてページ番号を追加
7. **印刷プレビュー**: 印刷プレビューで確認

### フォーマット関連

- `Ctrl+B`: 太字
- `Ctrl+I`: 斜体
- `Ctrl+U`: 下線

- `Ctrl+Alt+2`: 見出し2
- `Ctrl+Alt+3`: 見出し3

- `Ctrl+V`: 通常の貼り付け
- `Ctrl+Shift+V`: 形式を保持して貼り付け（Word 2016以降）
- `Ctrl+Alt+V`: 形式を選択して貼り付け
