## デプロイ前の必須確認

1. **SECRET_KEY** を本番用のランダム文字列に変更  
   → `python scripts/generate-secret-key.py` で生成
2. **CORS_ORIGINS** に本番フロントのURLを設定
3. **DATABASE_URL** を本番DB（PostgreSQL推奨）に設定
4. **REACT_APP_API_URL** を本番バックエンドURLに設定
5. **REACT_APP_SENTRY_DSN** を設定（エラー監視用・推奨）

詳細: `scripts/本番デプロイ前チェック.md`  
手順: `scripts/本番デプロイ手順.md`
