# スキル向上ロードマップ

UEP v5.0 参画者が習得・強化を目指すスキル一覧。資格等は条件除外。実務・個人開発・学習リソースで習得を目指す。

---

## 高優先度（実務でよく使う）

| スキル | 伸ばし方 |
|--------|----------|
| **Kubernetes** | 本番運用、HPA/VPA、NetworkPolicy、ResourceQuota、オペレーター |
| **Terraform** | モジュール化、state管理、マルチクラウド対応 |
| **Prometheus/Grafana** | SLO/SLI設計、アラート設計、長期保存、Thanos/Cortex |
| **Python** | 非同期処理、型ヒント、テスト、リファクタリング |
| **CI/CD** | マルチステージ、キャッシュ、セキュリティスキャン、シークレット管理 |

## 補強スキル（5スキル）

| スキル | 伸ばし方 |
|--------|----------|
| **Go** | インフラツール・CLI開発、Kubernetesオペレーター、並行処理（goroutine） |
| **eBPF** | 観測性（可観測性ツール）、ネットワーク、セキュリティ（Falco等） |
| **GraphQL** | API設計、N+1対策、フェデレーション、Apollo/Relay |
| **分散トレーシング** | OpenTelemetry、Jaeger、トレース設計・運用、コンテキスト伝搬 |
| **イベント駆動** | Kafka、Kafka Streams、イベントソーシング、CQRS、Sagaパターン |

### 補強スキル 詳細

| スキル | 学習リソース・実践例 |
|--------|----------------------|
| **Go** | A Tour of Go、Effective Go、Kubernetesオペレーター開発、CLIツール（cobra） |
| **eBPF** | Cilium、Falco、BCC、bpftrace、観測性ツール（Pixie等） |
| **GraphQL** | Apollo Server、Strawberry（Python）、N+1対策（DataLoader）、スキーマ設計 |
| **分散トレーシング** | OpenTelemetry SDK、Jaeger、トレース・スパン設計、W3C Trace Context |
| **イベント駆動** | Kafka Streams、イベントソーシング、CQRS、Saga、アウトボックスパターン |

## 学習の進め方

1. **高優先度** … 実務・個人開発で毎日触れ、深める
2. **補強スキル（5スキル）** … Go、eBPF、GraphQL、分散トレーシング、イベント駆動を興味・案件に応じて段階的に取り入れる
