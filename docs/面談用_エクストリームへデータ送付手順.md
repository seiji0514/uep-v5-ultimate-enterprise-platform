# エクストリームへ自分のデータを送る手順

※面談（2/28 実施済み）で、**先方からデータ添付の希望あり**。以下を送付する。

---

## 送付するデータ（面談時の要望に対応）

| 資料 | ファイル | 形式 |
|------|----------|------|
| 職務経歴書 | `docs/職務経歴書_株式会社エクストリーム_インフラエンジニア_小川清志.md` | Word/PDF |
| ポートフォリオ | `docs/ポートフォリオ_エクストリーム応募用.html` | PDF（印刷→保存） |

**送付方法**: type は書類添付ができないため、**こちら（小川）からメールで添付して送付**する。面談時に案内された連絡先・メールアドレスを使用する。

---

## 送付手順

### 1. PDF を作成する

| 資料 | 手順 |
|------|------|
| **職務経歴書** | `職務経歴書_株式会社エクストリーム_インフラエンジニア_小川清志.md` を Word で開く → 別名保存で PDF。または Markdown を PDF に変換するツールを使用 |
| **ポートフォリオ** | `ポートフォリオ_エクストリーム応募用.html` をブラウザで開く → Ctrl+P（印刷）→ 送信先「PDFに保存」 |

### 2. メールで送付する

- **宛先**: 面談時に案内された宛先（採用担当者メールアドレス等）
- **件名例**: 「【面談】小川清志・インフラエンジニア・職務経歴書・ポートフォリオ送付」
- **本文例**: 本日はありがとうございました。ご依頼の資料を添付いたします。よろしくお願いいたします。
- **添付**: 職務経歴書・ポートフォリオの PDF を添付

### 3. 宛先が不明な場合

- type メッセージで「先日はありがとうございました。type は書類添付ができないため、メールでお送りしたいのですが、送付先のメールアドレスをお教えいただけますか」と問い合わせる

- 障害説明書が必要な場合は `docs/障害説明書_エクストリーム_インフラエンジニア_小川清志.md` を同様に PDF 化して添付
