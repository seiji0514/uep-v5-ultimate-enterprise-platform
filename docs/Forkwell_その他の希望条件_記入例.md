## コピー用（Markdown対応）

```
## 譲れないこと
- **フルリモート（完全在宅勤務）**：身体障害者手帳保有のため、通院等の配慮をいただける環境で完全リモート勤務を希望します。通院は月1〜2回程度です。
- **完全リモートで健常者と同等以上のパフォーマンス**を発揮できる実績があります（約3年9ヶ月の完全在宅勤務経験）。

## できれば実現したいこと
- リモートワーク（週5日以上・完全在宅）
- スケジュール管理ツールでの通院予定の事前共有
- 案件選択制・単価連動など、納得感を持って働ける環境

## 得意なこと
- AI/ML技術（Python、RAG、LangChain、MLOps）
- フルスタック開発（設計・実装・テスト・運用まで一貫）
- アクセシビリティ対応のUI/UX設計
- 完全リモート環境での自律的な業務推進

## その他
- オフィス出社が必須の場合は、通院スケジュールと調整の上、月1回程度の出社が可能な場合があります（要相談）。
```
