【図8】適応的融合戦略の選択ロジック
開始
タスクタイプ、モーダル数、モーダル間の相関関係を取得
↓
判定1
タスクタイプは「分類」か？
Yes → Confidence Based戦略
No → 次の判定へ
↓
判定2
タスクタイプは「回帰」か？
Yes → Weighted Average戦略
No → 次の判定へ
↓
判定3
タスクタイプは「物体認識」か？
Yes → Context Aware戦略
No → 次の判定へ
↓
判定4
モーダル数は5以上か？
Yes → Hierarchical戦略
No → 次の判定へ
↓
判定5
モーダル間に強い相関が存在するか？
Yes → Attention Based戦略
No → Weighted Average戦略（デフォルト）
↓
終了
選択された融合戦略を適用
選択ロジックの説明
タスクタイプ、モーダル数、モーダル間の相関関係などの条件に基づいて、
Weighted Average、Confidence Based、Context Aware、Hierarchical、Attention Basedの
5つの融合戦略から適切な戦略を選択する。
