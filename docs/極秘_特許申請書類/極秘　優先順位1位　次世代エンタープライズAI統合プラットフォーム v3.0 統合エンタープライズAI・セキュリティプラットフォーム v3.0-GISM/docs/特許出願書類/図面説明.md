## 【図面の作成方法】

各図面は、以下の方法で作成する。

- **図1-6, 10**: ブロック図、フローチャート、シーケンス図は、Microsoft Visio、Draw.io、またはPlantUMLを使用して作成する。
- **図面の形式**: PDF形式または高解像度PNG形式（300dpi以上）で保存する。
- **図面のサイズ**: A4サイズ（210mm × 297mm）を標準とする。
- **図面の文字**: 日本語または英語で記載し、フォントサイズは10pt以上とする。
