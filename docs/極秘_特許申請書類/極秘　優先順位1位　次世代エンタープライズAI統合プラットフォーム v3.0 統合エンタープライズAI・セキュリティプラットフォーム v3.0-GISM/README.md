## 🚀 クイックスタート

### **セットアップ**

```bash
# 仮想環境の作成
python -m venv venv

# 仮想環境の有効化
venv\Scripts\activate  # Windows
source venv/bin/activate  # Linux/Mac

# 依存パッケージのインストール
pip install -r requirements.txt
```

### **アプリケーションの起動**

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

または、バッチファイルを使用：

```bash
3_アプリ起動.bat
```

### **テストの実行**

```bash
pytest tests/test_integrated_platform.py -v
```
