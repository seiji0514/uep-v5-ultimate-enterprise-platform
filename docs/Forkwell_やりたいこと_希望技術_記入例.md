## 1. やりたい・目指したいこと（コピー用）

```
AI/ML技術とアクセシビリティの両方に精通したエンジニアとして、誰もが使いやすいシステムの開発に貢献したいと考えています。

## 詳細
- **生成AI・LLM統合**を中心に、RAG、CoT推論、MLOps/LLMOpsの実装でクライアントの課題解決に貢献したい
- **フルスタック開発**（設計・実装・テスト・運用まで一貫）で、エンドツーエンドの品質を担保したい
- **アクセシビリティ対応**のUI/UX設計に、身体障害者としての当事者視点と専門知識を活かしたい
- **完全リモート環境**で、自律的に業務を推進し、健常者と同等以上のパフォーマンスを発揮したい
- 案件選択制・単価連動など、**納得感を持って働ける環境**で、技術力を最大限に発揮したい
```
