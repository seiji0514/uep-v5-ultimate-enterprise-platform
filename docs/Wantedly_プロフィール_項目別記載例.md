---

## 2. 職務概要（冒頭）

```
全職種対応エンジニアとして、Webサービスの設計・構築・運用経験を有し、システム全体の信頼性向上に貢献。フルリモート3年9ヶ月の実績。チーム開発での協働を通じて、データドリブンな意思決定とセキュリティ強化にも取り組みたい。
```

---

## 3. 【仕事を通じて実現したい価値】

```
5年間の台風対応統括経験を活かし、防災・法務・セキュリティ領域でデータドリブンな意思決定により人々の安全・安心を実現したいです。
```

---

## 4. 【なぜこれらの職種で働きたいか】

```
統合開発・運用プラットフォーム(IDOP)では、110以上のAPIエンドポイントを実装し、稼働率99.995%を達成。エンタープライズMLOps基盤では、Kubernetes、自動スケーリング、監視オーバーヘッド90%削減を実現。統合セキュリティコマンドセンターでは、自動対応率85%以上を実装。

これらの経験を活かし、チーム開発での実践を通じて、全職種対応エンジニアとして、システム全体の信頼性向上に貢献したい。
```

---
