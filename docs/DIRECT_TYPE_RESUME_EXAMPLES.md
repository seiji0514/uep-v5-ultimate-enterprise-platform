## 📋 Direct typeの入力フォーム構成

### 入力フィールド

- **従業員数**: 従業員数を入力
- **雇用形態**: 正社員、派遣社員、アルバイトなど
- **年収**: 年収を入力（任意）
- **事業概要**: 企業の事業概要を記載
- **配属先**: 配属先を記載
- **職種**: 職種を記載
- **職務内容**: 職務内容を詳細に記載
