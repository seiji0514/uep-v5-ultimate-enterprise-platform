## 11. 注意事項

1. **既に push 済みの場合**: `.gitignore` に追加しても、既存のコミット履歴には残ります。`git filter-repo` 等で履歴から削除する必要があります。
2. **Forkwell / type 職務経歴**: 汎用テンプレートのみ残し、企業名・個人情報入りは除外。
3. **docs/極秘_特許申請書類**: 最優先で除外。特許出願前の機密情報を含みます。
