# GDPR 対応ポリシー（設計）

**UEP v5.0 Level 5 - グローバルエンタープライズ**  
**対象**: EU一般データ保護規則（GDPR）対応の設計・実装方針

---

## 1. データ主体の権利

| 権利 | 対応設計 |
|------|----------|
| アクセス権（Article 15） | API: `/api/v1/compliance/data-access-request` |
| 訂正権（Article 16） | データ更新API、監査ログ |
| 削除権（忘れられる権利、Article 17） | 論理削除、物理削除プロセス |
| 処理制限権（Article 18） | フラグによる処理停止 |
| データポータビリティ（Article 20） | JSON/CSV エクスポートAPI |
| 異議申立権（Article 21） | オプトアウト設定、処理停止 |

## 2. 同意管理

- 同意取得時の記録（タイムスタンプ、バージョン）
- 同意撤回プロセス
- 同意の粒度（目的別）

## 3. データ処理の記録（Article 30）

- 処理目的
- データカテゴリ
- 受領者
- 保存期間
- 技術的・組織的措置

## 4. データ保護影響評価（DPIA）

- 高リスク処理の事前評価
- 記録の保持
