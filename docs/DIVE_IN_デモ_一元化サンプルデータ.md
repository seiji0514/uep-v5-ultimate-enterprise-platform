## 4分割構成（企業向けデモ）

| No  | カテゴリ                 | 含めるシステム                                                           |
| --- | ------------------------ | ------------------------------------------------------------------------ |
| 1   | **インフラ基盤**         | クラウドインフラ、インフラ構築専用                                       |
| 2   | **AI・ML**               | MLOps、生成AI、AI支援開発                                                |
| 3   | **監視・セキュリティ**   | セキュリティコマンドセンター                                             |
| 4   | **開発・運用**           | IDOP                                                                     |

## 概要

MLOps から Level 5 グローバルエンタープライズまで、**共通プロジェクトID**で紐づいたサンプルデータをバックエンド起動時に自動投入します。

| プロジェクトID   | プロジェクト名    |
| ---------------- | ----------------- |
| uep-demo-unified | UEP推論最適化基盤 |

---

## 投入されるデータ

### 1. MLOps（実験追跡・モデルレジストリ・パイプライン）

| 項目           | 内容                                        |
| -------------- | ------------------------------------------- |
| **パイプライン** | UEP推論最適化基盤 学習パイプライン（成功）、A/Bテスト モデル比較パイプライン（実行中） |
| **実験**       | 推論レイテンシ最適化、A/Bテスト モデル比較  |
| **モデル**     | sentiment-inference-v1（v1.2 Production）   |
| **メトリクス** | レイテンシ8.5ms、スループット1200、精度0.95 |

### 2. クラウドインフラ

| リソース                  | リージョン     | 説明                        |
| ------------------------- | -------------- | --------------------------- |
| uep-inference-eks-cluster | ap-northeast-1 | EKSクラスタ（Level 5 東京） |
| uep-mlops-s3-bucket       | ap-northeast-1 | モデルアーティファクト用S3  |
| uep-api-gateway           | ap-northeast-1 | Kong API Gateway            |

### 3. IaCテンプレート

- UEP EKS クラスタ（Level 5 東京）

### 4. デプロイメント（Kubernetes）

- uep-inference-api（3レプリカ）
- uep-prometheus（2レプリカ）

### 5. インフラ構築専用

- **UEP推論最適化基盤** - MLOps推論モデル→クラウドインフラ→Level 5東京リージョンまで一元化

### 6. Level 5 グローバルエンタープライズ

- 既存の config ベースデータ（東京・バージニア・アイルランド リージョン）と連携

---

## データの流れ（デモ説明用）

```
MLOps 実験「推論レイテンシ最適化」
    ↓ モデル v1.2 登録
モデルレジストリ sentiment-inference-v1
    ↓ デプロイ
クラウドインフラ（ap-northeast-1）
    ↓ EKS / Kubernetes
Level 5 東京リージョン
```

---

## 動作確認

バックエンド起動後、以下でデータが表示されます。

- **MLOps**: `/mlops` → 実験一覧・モデル一覧
- **クラウドインフラ**: `/cloud-infra` → リソース・テンプレート・デプロイメント
- **インフラ構築専用**: `/infra-builder` → プロジェクト一覧（UEP推論最適化基盤を追加）
- **グローバル (Level 5)**: `/global-enterprise` → マルチリージョン・東京
