# 転職ドラフトスカウト - キャリアビジョンの記載方法

**作成日**: 2026年1月30日  
**目的**: 転職ドラフトスカウトのプロフィール欄で「キャリアビジョン」を記載する際の判断基準

---

## 📋 記載項目

### 1. キャリアビジョンについて教えてください（必須）

**会社での実現に限定せず、個人的なものでもかまいません。**

例: アプリ開発で、家族を養えるくらいの生計を立てられるようになりたい。

### 2. そう思う理由はなんですか？具体的にはどんなことがしたいですか？

**キャリアビジョンの理由と具体的な行動計画**

---

## 🎯 ユーザーの状況整理

### 現在の状況

- **職務**: AI技術研究開発（2025年6月～現在）
- **希望勤務形態**: フルリモート（完全在宅勤務）
- **通院の配慮**: 必要
- **資格**: 身体障害者手帳保有、情報処理安全確保支援士試験 受験経験
- **技術的能力**: 技術的作業には全く支障がない、完全リモート環境であれば健常者と同等以上のパフォーマンスを発揮できる

### 実績・スキル

- **UEP v5.0**: エンタープライズレベルの統合プラットフォームを実装
- **技術スタック**: Python, FastAPI, React, TypeScript, AI/ML
- **実装範囲**: 7つのコアシステムの統合、本番環境対応の実装

## 💡 キャリアビジョンの提案

### 提案1: AI技術リーダーとしてのキャリアビジョン ⭐⭐⭐⭐⭐

#### キャリアビジョン

```
AI技術を活用したエンタープライズレベルのシステム開発で、企業のIT戦略課題を解決し、
技術的リーダーとしてチームを牽引できるようになりたい。
```

#### そう思う理由はなんですか？具体的にはどんなことがしたいですか？

```
【理由】
現在、AI技術研究開発に従事しており、UEP v5.0のようなエンタープライズレベルの統合プラットフォームを実装してきました。
AI技術の可能性を実感し、企業のIT戦略課題を解決する技術的リーダーとして成長したいと考えています。

【具体的にしたいこと】
1. AI技術を活用したエンタープライズシステムの設計・実装
   - MLOps基盤システム、生成AIシステムなどの統合プラットフォームの構築
   - 本番環境対応の実装、セキュリティ・スケーラビリティの確保

2. 技術的リーダーとしての役割
   - チームの技術的方針の決定・推進
   - コードレビュー、技術的負債の返済、技術的負債返済に理解のある環境での活動

3. 完全リモート環境での実績創出
   - 完全リモート環境であれば、健常者と同等以上のパフォーマンスを発揮できることを実証
   - リモートワークでの開発効率向上、チーム連携の最適化

4. 継続的な技術的成長
   - 最新のAI技術の習得・実装
   - エンタープライズレベルの技術スタックの習得・実装
   - 技術書購入支援、社内勉強会などの学習環境の活用
```
