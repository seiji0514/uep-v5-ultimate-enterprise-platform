# Google Sites でポートフォリオを作成する手順

## 概要

Google Sites はブラウザだけで作成できるウェブサイトビルダーです。ドラッグ＆ドロップでテキスト・画像・表を配置できます。HTML の直接アップロードはできません。

---

## 手順

### 1. サイトを作成

1. [Google Sites](https://sites.google.com) にアクセス
2. Google アカウントでログイン
3. **「＋ 空白」** または **「テンプレートから作成」** をクリック
4. 左上の **「無題のサイト」** をクリック → タイトルを **「ポートフォリオ - 小川 清志」** に変更

---

### 2. 概要セクション

1. 最初のテキストブロックに以下を入力:

```
氏名: 小川 清志
希望職種: インフラエンジニア / 全職種対応エンジニア

【概要】
インフラエンジニアとしての実装実績をまとめたものです。UEP v5.0 を中心に、Docker・Kubernetes・Terraform・監視基盤・MLOps 等の実績を記載しています。

インフラを中心に、バックエンド・フロントエンド・AIまで一貫して担当可能です。
```

2. 左の **「挿入」** パネル → **「表」** を選択して追加

---

### 3. 主要プロジェクト一覧（表）

1. 表を挿入 → 3列×4行に設定
2. 以下のように入力:

| プロジェクト | 概要 | 技術スタック |
|-------------|------|-------------|
| UEP v5.0 | 次世代エンタープライズ統合プラットフォーム | Docker, K8s, Terraform, Prometheus, Grafana, Kafka |
| IDOP | 統合開発・運用プラットフォーム | CI/CD, パイプライン, IaC |
| インクルーシブ雇用AI | 障害者雇用マッチング・アクセシビリティAI | LLM, RAG, マッチングエンジン |

---

### 4. UEP v5.0 セクション

1. **「挿入」** → **「テキスト」** で見出しを追加: **「1. UEP v5.0（次世代エンタープライズ統合プラットフォーム）」**
2. 本文: 「実装完了: 2026年1月　成果: 稼働率99.995%、推論レイテンシ10ms未満」
3. **「挿入」** → **「画像」** → **「アップロード」**
   - `docs/portfolio_images/01_login.png` をアップロード
   - キャプション: 「図1: ログイン画面（React・TypeScript・Material-UI）」
4. 同様に 02_dashboard.png、03_swagger.png を追加
5. 技術スタック: 「Docker, Kubernetes, Terraform, Prometheus, Grafana, Kafka, FastAPI, React, TypeScript」

---

### 5. インクルーシブ雇用AI セクション

1. 見出し: **「2. インクルーシブ雇用AI」**
2. 画像: `04_inclusive.png` をアップロード
3. 本文: 「障害者雇用マッチング・アクセシビリティ特化AI・当事者視点UX評価。Webアクセシビリティ検査員検定（レベル1～3）を活かした設計。」

---

### 6. その他の実績

1. 見出し: **「3. その他の実績」**
2. 箇条書きで追加:
   - 負荷テスト: Locust で実施、総リクエスト226・失敗0回・平均レスポンス約56ms
   - 100システム以上: 8ヶ月で実装（インフラ・監視・MLOps 含む）

---

### 7. オンラインプロフィール

1. 見出し: **「オンラインプロフィール」**
2. リンク付きテキストを追加:
   - LAPRAS: https://lapras.com/public/AEKCECT
   - Zenn: https://zenn.dev/seiji_ogawa
   - Qiita: https://qiita.com/seiji0514

※URL を入力すると自動でリンクになります。

---

### 8. 公開

1. 右上の **「公開」** をクリック
2. **ウェブアドレス** を選択
3. URL を設定（例: `portfolio-ogawa`）→ `https://sites.google.com/view/portfolio-ogawa` のような形式
4. **「公開」** をクリック

---

## 公開 URL の形式

```
https://sites.google.com/view/<サイト名>
```

この URL を Wantedly のプロフィール「リンク」欄に登録してください。

---

## 注意点

| 項目 | 内容 |
|------|------|
| **画像** | `docs/portfolio_images/` の4枚を Google Sites にアップロードする必要があります |
| **編集** | 公開後も「編集」から内容を変更できます |
| **無料** | 個人利用は無料です |
| **レスポンシブ** | スマホでも自動で見やすく表示されます |
