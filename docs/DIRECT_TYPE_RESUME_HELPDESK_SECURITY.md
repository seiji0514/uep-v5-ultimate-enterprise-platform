## 📋 警備業務（2009年11月～2014年12月）

**簡潔に記載、ただし災害対応の経験は記載**

### Direct typeの入力フォーム記載例

```
【従業員数】
[企業の従業員数を記載]

【雇用形態】
アルバイト

【年収】
[年収を記載、任意]

【事業概要】
[企業の事業概要を記載]

【配属先】
[配属先を記載]

【職種】
警備員（台風対策通行止判断の統括、高速道路交通誘導警備業務、雑踏警備業務）

【職務内容】
警備業務に従事。台風対策通行止判断の統括（最終判断責任者）、高速道路交通誘導警備業務、雑踏警備業務を担当。

【主な業務】
- 台風対策通行止判断の統括（最終判断責任者）
  - 平戸大橋・生月大橋の台風接近時の通行止判断を統括
  - 気象情報、風速、橋の状態等を総合的に判断し、通行止の最終決定を実施
  - 人命に関わる重要な判断を任される責任あるポジション
- 高速道路交通誘導警備業務（西九州道路・長崎道・SA）
- 雑踏警備業務（花火大会・イベント・大塔イオンの年末年始駐車場警備）

【成果】
- 台風対策通行止判断の統括を5年間担当
- この5年間の台風対応統括経験をシステム化した「災害対応統合コマンドセンター」を開発
```
