# UEP v5.0 改善前後の比較

面談用・本番化対応を含む（2026年2月〜3月）

---

## UX/UI

| 項目 | 改善前 | 改善後 |
|------|--------|--------|
| サイドバー | 320px固定 | 折りたたみ可能（72px/320px） |
| ナビゲーション | 現在位置不明 | パンくずリスト表示 |
| テーマ | ダークのみ | ダーク/ライトモード切替、localStorage永続化 |
| モバイル | 検索しづらい | 検索アイコン→ダイアログで最適化 |

## 機能拡張

| 項目 | 改善前 | 改善後 |
|------|--------|--------|
| 検索 | なし | グローバル検索（GlobalSearch） |
| ナビ | なし | お気に入り・最近使ったページ（ピン留め） |
| 通知 | なし | 通知センター、トースト通知 |
| 多言語 | 日本語のみ | i18n・言語切替対応 |

## パフォーマンス・品質検証

| 項目 | 改善前 | 改善後 |
|------|--------|--------|
| 読み込み | 全ページ一括 | React.lazy + Suspense でルート単位コード分割 |
| API | 都度取得 | React Query キャッシュ（staleTime 2分等） |
| 計測 | なし | Web Vitals 計測 |
| メモリ | ビルド時不足の可能性 | NODE_OPTIONS=8192、GENERATE_SOURCEMAP=false |
| 負荷テスト | 未実施 | Locust 実施（2026年2月19日）総226リクエスト・失敗0・平均56ms |

## 堅牢性

| 項目 | 改善前 | 改善後 |
|------|--------|--------|
| エラー | 個別対応 | グローバルエラーバウンダリ、再試行UI |
| オフライン | 未対応 | Service Worker、OfflineBanner |
| 空データ | 表示バラバラ | EmptyState コンポーネント統一 |
| ローディング | 一部のみ | LoadingSkeleton、フォームバリデーション |

## セキュリティ・本番化

| 項目 | 改善前 | 改善後 |
|------|--------|--------|
| CSP | 未設定 | CSP設定、Sentry許可 |
| セッション | 期限切れでログアウト | リフレッシュトークン自動延長、5分前警告 |
| 監査 | バックエンド中心 | 監査ログAPI永続化、login/logout記録 |
| 監視 | なし | Sentry エラー収集、ErrorBoundary連携 |
| レート制限 | なし | ログイン 10回/分 |

## アクセシビリティ

| 項目 | 改善前 | 改善後 |
|------|--------|--------|
| キーボード | 一部未対応 | Tab・Enter・Space で操作可能 |
| ARIA | 不足 | aria-label、aria-busy 追加 |
| フォーカス | 薄い | 2px outline で視認しやすく |

## バックエンド

| 項目 | 改善前 | 改善後 |
|------|--------|--------|
| datetime | utcnow()（DeprecationWarning） | datetime.now(timezone.utc) 一括置換 |
| ヘルスチェック | /health のみ | /health/detailed、/api/v1 バージョン情報 |

## 開発者体験

| 項目 | 改善前 | 改善後 |
|------|--------|--------|
| E2E | 不足 | dashboard.spec.ts |
| モック | 本番API依存 | MSW |
| コンポーネント | なし | Storybook |
