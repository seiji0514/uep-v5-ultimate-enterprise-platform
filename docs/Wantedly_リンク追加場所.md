## Wantedly での追加場所

### 1. プロフィール編集画面を開く

1. Wantedly にログイン
2. 右上のプロフィールアイコン → **プロフィールを編集**
3. または https://www.wantedly.com/id/kiyoshi_ogawa_c の **編集** ボタン

### 2. リンクを追加する項目

Wantedly のプロフィール編集画面で、以下のいずれかに該当する項目を探す：

| 項目名（例） | 追加内容 |
|-------------|----------|
| **URL** / **リンク** / **ウェブサイト** | LAPRAS, Zenn, Qiita, ポートフォリオのURL |
| **SNS** | 上記リンク（SNS欄がある場合） |
| **自己紹介** | 本文中に「LAPRAS: [URL]」「Zenn: [URL]」等を記載 |

※Wantedly の画面構成は変更される場合があります。編集画面内で「URL」「リンク」「ウェブサイト」「SNS」などの項目を探してください。

### 3. 自己紹介にリンクを書く場合

自己紹介文の末尾に追記：

```
【リンク】
LAPRAS: https://lapras.com/public/AEKCECT
Zenn: https://zenn.dev/seiji_ogawa
Qiita: https://qiita.com/seiji0514
ポートフォリオ: （アップロード後のURL）
```

## ポートフォリオの公開方法（Wantedly 用）

汎用ポートフォリオ（`docs/ポートフォリオ_汎用.html`）を Web で公開する方法：

| 方法 | 手順 |
|------|------|
| **GitHub Pages** | 詳細は **GitHub_Pages_ポートフォリオ公開手順.md** を参照 |
| **Notion** | 詳細は **Notion_ポートフォリオ作成手順.md** を参照 |
| **Google Sites** | 詳細は **Google_Sites_ポートフォリオ作成手順.md** を参照 |
| **Google Drive** | HTML をアップロード → 共有リンク（HTML の表示は制限あり） |
| **Surge.sh** | 詳細は **Surge_ポートフォリオ公開手順.md** を参照 |
| **Netlify / Vercel** | 静的サイトとしてデプロイ |

※画像（portfolio_images）を含む場合は、同じディレクトリ構成でアップロードする必要があります。
