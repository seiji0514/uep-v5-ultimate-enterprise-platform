# Direct type（ダイレクトタイプ） - 職務経歴書アップロードエラー対処法

**作成日**: 2026年1月30日  
**目的**: Direct typeで職務経歴書がアップロードできない場合の対処法

---

## ⚠️ エラーの原因

### エラーメッセージ

**「※「▲▽」が入力フォームに残っていると、保存することができません。」**

### 原因

**入力フォームに「▲▽」という文字が残っている**

- ⚠️ **プルダウンメニューから雛形を呼び出した際**: 選択が完了していない状態でプレースホルダーやマーカーが残っている
- ⚠️ **「職務内容」フィールド**: 「職務内容」フィールドに「▲▽」が残っている可能性が高い
- ⚠️ **コピー&ペースト**: 他の場所からコピー&ペーストした際に「▲▽」が含まれている

---

## ✅ 解決方法

### 方法1: 「▲▽」を削除する（最も簡単）

**「職務内容」フィールドから「▲▽」を削除**

#### 手順

1. **「職務内容」フィールドを確認**: 「職務内容」フィールドの内容を確認
2. **「▲▽」を検索**: 「▲▽」という文字を検索（Ctrl+F）
3. **「▲▽」を削除**: 「▲▽」を削除
4. **保存**: 「保存して戻る」ボタンをクリック

---

### 方法2: 雛形を正しく選択する

**プルダウンメニューから雛形を呼び出した場合、正しく選択する**

#### 手順

1. **雛形を呼び出す**: 「雛形を呼び出す」ボタンをクリック
2. **雛形を選択**: プルダウンメニューから適切な雛形を選択
3. **「▲▽」を削除**: 選択後、「▲▽」が残っている場合は削除
4. **保存**: 「保存して戻る」ボタンをクリック

---

### 方法3: テキストを手動で入力する

**雛形を使わずに、テキストを手動で入力する**

#### 手順

1. **「職務内容」フィールドをクリア**: 「職務内容」フィールドの内容を全て削除
2. **テキストを手動で入力**: テキストを手動で入力
3. **「▲▽」が含まれていないか確認**: 「▲▽」が含まれていないか確認
4. **保存**: 「保存して戻る」ボタンをクリック
