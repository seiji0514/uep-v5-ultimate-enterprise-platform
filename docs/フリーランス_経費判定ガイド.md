# フリーランス 経費判定ガイド

自分用の経費・非経費の判別の目安です。最終判断は税理士・税務署にご確認ください。

---

## 経費になる（○）

| カテゴリ | 例 | 備考 |
|----------|-----|------|
| **通信費** | インターネット料金、スマホ（業務用割合） | 自宅作業なら按分（例：50%） |
| **PC・周辺機器** | パソコン、モニター、キーボード | 10万円超は減価償却 |
| **ソフト** | 開発ツール、クラウドサービス、Adobe | 業務で使うもの |
| **書籍** | 技術書、資格関連の本 | 業務に直結するもの |
| **セミナー・勉強会** | 参加費、オンライン講座 | スキル向上・業務関連 |
| **交通費** | 打ち合わせ、出張の往復 | 領収書・明細を保管 |
| **消耗品** | メモリ、USB、紙・インク | 業務で消費するもの |
| **接待交際費** | 取引先との打ち合わせ飲食 | 1人5,000円以下等の制限あり |
| **支払手数料** | 振込手数料、決済手数料 | 業務に伴う手数料 |
| **保険** | 業務用賠償保険 | 業務に必要な保険 |
| **家賃・光熱費** | 自宅の一部を事務所 | 按分（例：部屋の面積割合） |
| **コワーキング** | 利用料金 | 全額経費 |

---

## 経費にならない（×）

| カテゴリ | 例 | 理由 |
|----------|-----|------|
| **通勤** | 自宅⇔自宅の交通費 | 自宅が事務所の場合、通勤とみなされない |
| **私用** | プライベートの飲食、趣味 | 業務と無関係 |
| **生活費** | 食費、日用品（業務用でない） | 個人の生活費 |
| **医療費** | 病院、薬 | 経費ではなく医療費控除の対象 |
| **生命保険** | 個人の保険 | 生命保険料控除の対象 |
| **趣味の本** | 小説、娯楽 | 業務と無関係 |

---

## グレーゾーン（要判断）

| 例 | 目安 |
|-----|------|
| スマホ | 業務使用割合（例：50%）を按分して経費 |
| 自宅の家賃・光熱費 | 専用スペースがある場合、面積割合で按分可能（手続きやや複雑） |
| カフェでの作業 | 打ち合わせなら経費、一人作業のみは判断が分かれる |
| 在宅勤務時の昼食 | 通常は経費にしない（接待は別） |

---

## 判別の目安（3つの質問）

1. **業務で使ったか？** → はいなら経費の可能性大
2. **プライベートと混在するか？** → 按分が必要
3. **領収書・明細を取れるか？** → 取れないと経費にしにくい

---

## 注意

- 本ガイドは目安です。税務上は税理士または税務署にご確認ください。
- 領収書・明細は5〜7年保管が推奨です。
