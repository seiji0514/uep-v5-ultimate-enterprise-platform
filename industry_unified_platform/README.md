# 産業統合プラットフォーム

製造・IoT + 医療・ヘルスケア + 金融・FinTech + 統合セキュリティ・防衛 を**1つに統合**した個別システムです。

## 概要

- **バックエンド**: ポート 9010（製造・医療・金融・セキュリティの4つAPIを1つに統合）
- **フロントエンド**: ポート 3010（タブで切り替え）

## 起動方法

1. バックエンド: `start-backend.bat`
2. フロントエンド: `start-frontend.bat`
